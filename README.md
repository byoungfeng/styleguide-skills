# Google Style Guide Skills

All 17 Google open-source coding style guides, packaged as loadable skills.

## Usage

Each skill contains:
- `SKILL.md` — YAML front matter + quick reference card
- `references/deep.md` — in-depth reference with edge cases and review checklist

Load a skill by name when working on a matching coding task.
